## 安装

clone 项目, 将Tokyonight-Day 和 Tokyonight-Storm两个目录复制到fcitx5的主题目录中,在Arch linux下,该目录默认为`~/.local/share/fcitx5/themes`,打开fcitx5 configuration选用该主题即可
